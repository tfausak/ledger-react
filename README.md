# [ledger-react][1]

Watch your money fly away.

This is a front end for [Ledger][2].

- [Installation](#installation)

## Installation

To install ledger-react, you'll need Node.js v0.10.

``` sh
$ git clone https://github.com/tfausak/ledger-react.git
$ cd ledger-react
$ npm install
$ bower install
$ npm run start
# http://localhost:8000
```
